🏥 Doctor Appointment Booking System (MERN Stack)

This is a MERN Stack based web application for booking doctor appointments.

🚧 Project Status: Under Development

The project is currently in progress and some features are still being implemented.

🚀 Current Features Implemented
✅ Frontend

React setup with routing

Doctor listing page

Doctor details page

Appointment UI

User profile page (edit functionality)

Responsive layout with Tailwind CSS

✅ Backend

Express server setup

MongoDB connection

User model

Doctor model

Basic API routes

JWT authentication setup

⏳ Features In Progress

Appointment booking logic completion

Admin dashboard

Payment integration

Email notifications

Doctor availability time slots

Profile image upload to Cloudinary

Full appointment history

🛠 Tech Stack

Frontend

React.js

Tailwind CSS

Context API

Axios

Backend

Node.js

Express.js

MongoDB

Mongoose

JWT

⚙️ How to Run the Project
Backend
cd backend
npm install
npm run server

Frontend
cd frontend
npm install
npm run dev

📌 Goal of This Project

This project is being built to:

Practice full-stack MERN development

Understand authentication and protected routes

Learn file upload handling

Improve frontend design skills

Build a real-world appointment system
